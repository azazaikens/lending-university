Сучасний центр безпеки

Матеріально-технічна база кафедри

У лабораторії кафедри розташований ситуаційний центр з сучасним обладнанням та програмним забезпеченням для аналізу та обробки інформаційної безпеки. Використовуються SIEM-системи, програмні сканери Nessus і Kali Linux для пошуку вразливостей, а також методи управління ризиками на основі CRAMM, OCTAVE та інших методологій. Використовуються стандарти ISO 31000, ISO 27005 тощо.

Ситуаційний центр дозволяє візуалізувати ситуації інформаційної небезпеки та розвивати у студентів навички управління ризиками для гарантованого працевлаштування.

Знання для Захисту:

Розробка нашої кафедри для вас!

Науково-педагогічний колектив кафедри Управління інформаційною та кібернетичною безпекою розробив нову дисципліну «OSINT. Аналітика даних з відкритих джерел». Її метою є пошук, збір та аналітика даних у галузі інформаційних технологій, з фокусуванням на кібербезпеці.

У ній застосовуються найсучасніші інструменти для збору і аналізу інформації. Задля роботи з інструментами OSINT в аудиторії №401 було встановлено операційну систему Kali Linux, призначену для цифрової криміналістики і тестування на проникнення. Її встановили для роботи зі спеціальним програмним забезпеченням, яке використовується для збору інформації під час процесу оцінки інформаційної безпеки.
